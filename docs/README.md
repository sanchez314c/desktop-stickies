## ğŸš€ Quick Start

### For Users
1. Download StickyNotes from the [Mac App Store](https://apps.apple.com)
2. Open the app - it runs in the background
3. Press `Cmd+N` to create your first note
4. Start taking notes!

### For Developers
1. Clone the repository: `git clone https://github.com/your-org/stickynotes.git`
2. Install dependencies: `swift package resolve`
3. Open in Xcode: `open StickyNotes.xcodeproj`
4. Build and run: `Cmd+R`
