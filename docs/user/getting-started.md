### Quick Fixes

**Notes not appearing?**
- Check menu bar for StickyNotes icon
- Try `Cmd+N` to force create a note
- Restart StickyNotes

**Can't type in notes?**
- Click directly in the note area
- Make sure note is selected (blue border)

**App not responding?**
- Force quit: `Cmd+Option+Esc`
- Reopen from Applications folder
