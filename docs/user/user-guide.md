## ğŸ“ Creating and Managing Notes

### Creating Notes

**Method 1: Keyboard Shortcut**
- Press `Cmd+N` to create a new note instantly

**Method 2: Menu Bar**
- Click the StickyNotes icon in your menu bar
- Select "New Note"

**Method 3: Context Menu**
- Right-click on the desktop
- Select "New Sticky Note" (if enabled in preferences)

### Note Colors

StickyNotes supports 6 predefined color schemes:

- **Yellow** (Default) - Classic sticky note color
- **Blue** - Professional and clean
- **Green** - Nature-inspired
- **Pink** - Creative and fun
- **Purple** - Elegant and distinctive
- **Gray** - Minimal and subtle

**To change colors:**
1. Click the color indicator in the title bar
2. Select your preferred color from the dropdown

### Note Positioning and Sizing

**Moving Notes:**
- Click and drag the title bar to move notes around your screen
- Notes remember their position between sessions

**Resizing Notes:**
- Drag the bottom-right corner to resize
- Minimum size: 200x150 pixels
- Maximum size: Limited by screen dimensions

### Note Content

#### Basic Text Editing

- **Typing**: Click anywhere in the note and start typing
- **Selection**: Click and drag to select text
- **Copy/Paste**: Standard `Cmd+C` / `Cmd+V` shortcuts work
- **Undo/Redo**: `Cmd+Z` / `Cmd+Shift+Z`

#### Rich Text Formatting

StickyNotes supports rich text formatting:

- **Bold**: Select text and press `Cmd+B`
- **Italic**: Select text and press `Cmd+I`
- **Underline**: Select text and press `Cmd+U`
- **Strikethrough**: Select text and press `Cmd+Shift+X`

#### Lists and Structure

- **Bullet Lists**: Press `Cmd+Shift+8` or use the format menu
- **Numbered Lists**: Press `Cmd+Shift+7` or use the format menu
- **Checklists**: Use bullet lists with checkbox characters (â˜ â˜‘)

#### Markdown Support (Optional)

Enable Markdown mode in preferences for:
- `# Headers`
- `**Bold**` and `*italic*` text
- `- Bullet lists`
- `` `code` `` snippets
- `[Links](url)`

## ğŸ“¤ Exporting Notes

### Export Formats

StickyNotes supports multiple export formats:

- **Plain Text (.txt)**: Simple text format, preserves basic formatting
- **Rich Text (.rtf)**: Maintains fonts, colors, and formatting
- **Markdown (.md)**: Structured text with Markdown syntax
- **PDF (.pdf)**: Print-ready format with exact visual layout

### Export Methods

**Method 1: Individual Note Export**
1. Open the note you want to export
2. Click the "Export" button in the title bar
3. Choose your format and destination

**Method 2: Bulk Export**
1. Use the menu bar icon
2. Select "Export All Notes"
3. Choose format and destination folder

**Method 3: Quick Export**
- Press `Cmd+E` to export the current note
- Uses your last selected format and location
